Oropharyngeal Airway, In this asset pack you will find a Oropharyngeal Airway model with 9 different sizes of the model and 8 different colors of the model. This model has 760 polys and 4 PBR texture maps. The size of each map is 2048 pixels. This model has two shaders, One is for plastic and other one is for glass. Hence you can apply different shaders if you have it. This model is good for VR games, PC games, Mobile games and VR and AR simulations. You do not need any seperate shader or script to use this pack. Please use the prefabs so that you will not lose any shader or any maps.
